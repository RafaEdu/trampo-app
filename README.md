# trampo-app

<p>Aplicativo voltado para o público em geral, tanto para quem busca uma renda-extra ou até mesma uma forma difinitiva de trabalho, quanto para quem quer contratar um tipo especifico de especialista para um trabalho especifico.</p>

<p>De ínicio o objetivo é manter um aplicativo "free to use". AOnde é possível existir uma interação entre contratantes e contratadores.</p>
